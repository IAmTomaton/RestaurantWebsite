# RestaurantWebsite
Book a table - забронировать столик
Remove the reservation - удалить бронь
Reservations - список всех броней, символизирует то как хозяин получает список заказаов.

Tables - список всех столов
Create table - Создать слол
Delete table - Удалить стол

Как запустить
Ставим зависимости npm install
Запускаем npm run devstart
Переходим по ссылке http://localhost:3000/
